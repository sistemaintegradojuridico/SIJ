# SIJ - Sistema Integrado Jurídico

Sistema jurídico estilo ADVBOX com Kanban de processos.

## 🚀 Funcionalidades

- Criar processos
- Mover entre colunas (Novo, Andamento, Finalizado)
- Excluir processos
- Interface moderna (roxo vinho + dourado)

## 🛠 Tecnologias

- Node.js
- Express
- SQLite
- HTML, CSS, JavaScript

## ▶️ Como rodar

### Backend

cd backend  
npm install  
node server.js  

### Frontend

Abrir o arquivo:  
frontend/index.html
